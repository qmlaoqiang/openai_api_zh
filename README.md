# openai_api_zh
openAI API中文版
## 简介
### 概览
OpenAI的API可被用于任何涉及自然语言理解和生成的任务。我们提供一系列不同功能等级的模型以适配您各种场景，甚至于提供模型微调（fine-tune）的能力以让您可以充分定制您的模型。
这些模型可被用于从内容理解到语义搜索及分类的任何任务。
